Visión General: Centraliza la experiencia de compra y la operación interna de una tienda de moda urbana. El cliente gestiona su carrito y checkout adaptable, mientras que la administración controla el inventario, clientes, entregas y configuración general.

Arquitectura y Tecnologías

  • ASP.NET Core 9 MVC
  • Entity Framework Core 9
  • SQL Server
  • Razor / CSS / JS

Capacidades y Seguridad

  • Seguridad avanzada con FIDO2 / WebAuthn, aprobación de dispositivos y rate limiting.
  • Carrito persistente en sesión y flujos de "checkout" con instantáneas históricas.
  • Migraciones y utilidades para inyección de datos desde JSON.
  • Exportación administrativa de datos vía ClosedXML.
  • Mecanismos robustos: Antiforgery, cookies seguras y CSP con nonce.

Visión General: Plataforma web diseñada para reducir el trabajo manual y unificar la operativa de múltiples sedes, brindando a administradores, docentes y estudiantes las herramientas necesarias para su día a día.

Arquitectura y Tecnologías

  • ASP.NET MVC 5
  • .NET Framework 4.8
  • Entity Framework 6
  • Microsoft SQL Server

Capacidades y Seguridad

  • Facturación Electrónica y generación de XML/PDF con integraciones a Hacienda.
  • Control total de matrículas, cursos, evaluaciones y seguimientos mediante reportes gráficos.
  • Gestión de acceso mediante Perfiles y Roles (RBAC), con soporte de 2FA y WebAuthn.
  • Módulo de auditoría, recordatorios automatizados y comunicaciones integradas por correo.
